# Portfolio

Personal portfolio website built with Nuxt 4, Vue 3, and Tailwind CSS.

**Live site:** https://exiviuz.github.io/portfolio/

## Tech Stack

- [Nuxt 4](https://nuxt.com/) — Vue meta-framework
- [Vue 3](https://vuejs.org/) — UI framework
- [Tailwind CSS v4](https://tailwindcss.com/) — styling
- [VueUse](https://vueuse.org/) — composable utilities
- [Iconify](https://iconify.design/) — icons

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server at http://localhost:3000
npm run dev

# Build for production
npm run build

# Generate static site
npm run generate
```

## Deployment

Deployed via GitHub Pages using the static output from `nuxt generate`.
